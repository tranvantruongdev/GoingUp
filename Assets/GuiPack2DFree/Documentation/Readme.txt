Intro

GUIPack2DFree is a clean and minimalistic 2D UI pack designed to help you 
build a popup system for your current or next game. This mobile-friendly
pack includes a variety of components as prefabs, making it easy to create 
your own custom popup system.

- Visual effects: animations, sprites in the game
- Full C# source code included
- User customizable
- Works in Unity Personal and Unity Professional
- Simple and clean code
- Works on all platforms: PC, MacOS, iOS, Android, WebGL
- Optimized for mobile and tablet devices

Files structure:

1) Animations: animations which are used in project
2) Audio: sound file included in project
3) Documentation: documentation about the project
4) Fonts: fonts and fonts assets used in the project
5) Popups: prefabs, sprites and scripts of all available popups
6) Prefabs: prefabs used in project
7) Scenes: scenes used in project
8) Scripts: script files
9) Sprites: sprite files used in project
